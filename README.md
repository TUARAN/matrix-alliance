# 矩阵号联盟

> 把"个人写作"升级为"可复制的创作者工业体系"，让内容像零部件一样被复用、被分发、被结算。

## 项目简介

矩阵号联盟是一个创新的内容创作与分发平台，通过将内容生产部件化、模板化，构建完整的创作者生态体系。平台连接三大角色：生产者（矩阵号联盟）、放大者（博主联盟）和驱动者（系统中枢），实现内容从生产到分发再到结算的全链路数字化管理。

## 技术栈

- **前端框架**: Vue 3 + Vite
- **UI框架**: Tailwind CSS
- **路由**: Vue Router 4
- **状态管理**: Pinia
- **UI组件**: Headless UI
- **图表**: Chart.js + vue-chartjs
- **通知**: vue-toastification

## 功能模块

### 1. 首页 Hub
- 使命与价值展示
- 三大角色介绍
- 核心功能入口
- 数据战报展示

### 2. 任务市场
- 任务列表浏览
- 多维度筛选（类型、预算、难度）

- **AI助手**: 智能内容生成与优化

### 4. 分发中心
- 账号池管理
- 多平台分发编排
- 定时发布计划
- AB测试支持
- 分发数据追踪

### 5. 数据分析
- 关键指标展示（曝光、互动、分发等）
- 性能趋势分析
- 平台分布统计
- 内容表现排行
- ROI分析

### 6. 结算中心
- 账户余额管理
- 收益构成分析
- 结算记录查询
- 复用收益明细
- 提现与发票

### 7. 联盟中枢
- 创作者档案管理
- 账号资产绑定
- 内容部件目录
- 资质证明
- 合规协议管理

## 设计理念

### 三大角色

1. **生产者（矩阵号联盟）**: 提供可复用内容部件，标准化生产流程
2. **放大者（博主联盟）**: 负责分发与放大，多平台矩阵运营
3. **驱动者（系统中枢）**: 数据、工具、结算与风控

### 核心价值

- **部件化**: 内容像零部件一样被复用
- **模板化**: 标准化生产流程，提高效率
- **数据驱动**: 全链路归因，智能洞察
- **自动化**: 从任务到结算，全流程自动化

## MVP范围（90天）

- ✅ 任务市场（资讯快写/测评包两类）
- ✅ 部件化编辑器 + 模板库 v1
- ✅ 分发中心 v1（掘金/CSDN/公众号/小红书 4渠道）
- ✅ 数据回流 v1（基础看板）
- ✅ 结算 v1（复用计数 + 任务费用）
- ✅ 联盟中枢 v1（档案/账号绑定）
